Shortcuts
  • W/A/S/D or Arrow keys: Move
  • Q / E: Zoom out / Zoom in
  • Shift + F: Draw (Paint tool)
  • Shift + X: Delete tool
  • Shift + B: Select (Lasso)
  • Shift + S: Save
  • Shift + E: Eraser
  • Shift + P: Paint brush
  • Shift + R: Polygon pen
  • Shift + C: Bucket fill
  • Ctrl/Cmd + D: Deselect all
  • Ctrl/Cmd + Z: Undo
  • Ctrl/Cmd + Y or Shift+Ctrl/Cmd+Z: Redo
